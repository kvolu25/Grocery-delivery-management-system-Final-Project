# Grocery Delivery Management System

This project is a simple web application for managing grocery orders and deliveries.  
It allows customers to browse items, add them to a cart, and place orders.  
Managers can manage inventory and assign orders to drivers.  
Drivers can update delivery status.

---

## Features
- Browse grocery items
- Add items to cart
- Checkout and create orders
- Track order status
- Inventory management (add/update items)
- Assign orders to drivers
- Driver status updates

---

## Requirements
- Python 3.10+
- Flask
- See `requirements.txt` for dependencies
